# 未实现功能清单0

本文档只记录当前仍未真正实现、仍是占位、演示兜底或仅保存配置但没有业务闭环的功能。

## 传输内核

1. 真实 UDP 多线程传输调度与实时进度
   - 当前状态：部分未实现。`UdpTx` 和 `UdpRx` 已实现目标级并发 UDP 文件发送、停止等待 ACK、后台接收和接收目录落盘；`LocalBackend` 已让多个传输请求进入单线程 FIFO 队列；`UdpTx` 已在发送端按进度节点推送传输中快照，`FileTransfer` 会实时刷新发送进度；`UdpRx` 已在接收端按进度节点推送回调，`MainWindow` 会用 toast 展示接收进度。
   - 缺口：还没有分片级并发。
   - 推荐实现：只有在大文件性能不足时，再把单文件分片分派到并发 worker，并配套滑动窗口或缺失分片重传协议，避免 UDP 乱序导致误判失败。
